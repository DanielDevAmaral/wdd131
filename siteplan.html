Planify - Site Plan

Site Name: Planify

The site name was selected to bring to the mind of the user the urge to plan.

Site Purpose

The site allow the user to create tasks that will help him/her achive their objectives

Scenarios

Two important scenarios that the web site aims to solve:

  1. What I'm gonna do next?
  2. There is any important task that I need to remember?

Color Schema

The color scheme is based in a minimalistic style, the secondary and the third color jsut appears in some texts:

  • Primary Color: #000
  • Secondary Color: #D93239
  • Third Color: #7AC943

Typography

The chosen font for the website is Poppins
